For simple integration use {@link com.playphone.multinet.MNDirect MNDirect} class and {@link com.playphone.multinet.IMNDirectEventHandler IMNDirectEventHandler} interface only.

MultiNet Direct API (Simple)

MultiNet Direct API allows to add MultiNet functionality to application with mininal efforts. MultiNet Direct API is provided as a set of methods of {@link com.playphone.multinet.MNDirect MNDirect} class.
  • {@link com.playphone.multinet.MNDirect MNDirect}
  • {@link com.playphone.multinet.IMNDirectEventHandler IMNDirectEventHandler}

MultiNet connection key classes and interfaces (Advanced)

  • {@link com.playphone.multinet.MNSession MNSession}
  • {@link com.playphone.multinet.IMNSessionEventHandler IMNSessionEventHandler}

MultiNet user interface key classes and interfaces (Advanced)

  • {@link com.playphone.multinet.MNUserProfileView MNUserProfileView}
  • {@link com.playphone.multinet.IMNUserProfileViewEventHandler IMNUserProfileViewEventHandler}
